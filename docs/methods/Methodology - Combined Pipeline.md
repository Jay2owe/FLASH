# Image Analysis and Quantification — Combined Methodology

All image analysis was performed using the IHF Analysis Pipeline, a custom Fiji/ImageJ2 plugin developed for automated, reproducible quantification of immunohistofluorescence data. Multi-channel confocal z-stack images acquired in Leica .lif format were imported via Bio-Formats. Images were automatically oriented to a standard left-hemisphere reference based on filename convention (Experiment-AnimalID_Hemisphere_Region), and user-defined ROIs were applied to delineate the tissue area, with non-ROI regions cleared prior to analysis.

**3D object detection and quantification.** For cell counting and morphometric analysis, each fluorescence channel was pre-processed using configurable filter macros (e.g., median filtering, background subtraction). Objects were segmented in 3D using the channel's saved method: classical intensity thresholding followed by the 3D Objects Counter (mcib3d-core; Ollion et al., 2013), Enhanced Classical thresholding with object-level morphology filters, StarDist slice-wise segmentation with Z-linking (Weigert et al., 2020), Cellpose 3 segmentation, or a project-trained Smile Random Forest post-filter applied to a classical base segmentation. No watershed separation was applied in the classical path; touching objects were merged unless filter pre-processing, morphology filtering, StarDist, Cellpose, or the trained post-filter had already separated or rejected them. For each detected object, volume (µm³), surface area (µm²), integrated density, mean fluorescence intensity, and 3D centroid coordinates were recorded. Intensity measures were obtained from the unfiltered original channel image via redirection. Pairwise colocalization between all marker combinations was computed as the percentage volume overlap using the mcib3d colocalization algorithm. When applicable, neurite process length was extracted by skeletonizing the process channel, subtracting the nuclear marker, and quantifying skeleton length in calibrated units.

**Spatial analysis.** Inter-marker spatial relationships were assessed by computing 3D Euclidean nearest-neighbour distances between object centroids for all channel pairs within each tissue section. For each object, the distance to the closest object of every other marker was recorded, along with the number of objects for which it was the nearest neighbour and the number of colocalised nearest neighbours (colocalisation threshold: 30%). Where applicable, distances from detected objects to user-drawn anatomical reference lines were also computed as the minimum perpendicular distance from object centroids to polyline segments.

**Fluorescence intensity quantification.** For area-based intensity measurements, each channel was pre-processed with the selected filter source: either the per-channel filter macro saved in `.bin` or the built-in Basic background and noise removal filter (median radius = 2 pixels plus rolling-ball background subtraction, radius = 50 pixels). Where required, signal binarization was applied per channel at an explicit threshold, restricting intensity measurements to pixels above the threshold while preserving raw fluorescence values under the binary mask. Measurements could be confined to tissue ROIs and optionally masked by a binarized reference channel using that channel's selected filter and threshold. For each channel and z-plane, integrated density (mean × area), percent area (fraction of non-zero pixels), and raw integrated density (from the unfiltered image) were recorded.

**Data output.** All measures were exported as per-channel CSV files containing section identifiers, animal identifiers, and per-object or per-slice quantitative data for downstream statistical analysis.
